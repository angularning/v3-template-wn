

#   a project  template


## 介绍

本项目是基于Vue3 + Element-Plus + Vite 开发的实现后台管理系统。

#### 1.1、使用技术
Vite + Vue3 + Element-Plus + Sass + Pinia + Vue-router + Vue-i8n + Echarts
#### 1.2、兼容性注意
Vite 需要 Node.js 版本 >= 16.0.0。



## 项目运行


`pnpm install `

`pnpm run dev`
